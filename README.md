# Сайт Николая Шеламова — shelamov.ru

Статический сайт поэта Николая Шеламова, размещён на **GitHub Pages**.
Перенесён с Google Sites: 170 стихотворений в 4 сборниках, разделы «Об авторе»
и «Электронные книги».

## Превью
До переключения домена сайт доступен по адресу:
**https://5r4rw6n9n2-svg.github.io/**

## Как устроено
- Готовые статические HTML-файлы лежат в корне (`index.html`, `st/…`, `ob-avtore/…`).
  Это **источник истины** — их можно править вручную.
- Оформление — `assets/style.css` (классический литературный стиль).
- `assets/img/portrait.jpg` — портрет автора.
- SEO: `sitemap.xml`, `robots.txt`, canonical/OpenGraph/JSON-LD в каждой странице.
- Структура URL повторяет прежний сайт 1:1, плюс редирект `/news → /`.

## Правка контента
1. **Поправить один стих/страницу** — открыть нужный `index.html` и отредактировать.
   - Отступ строки («лесенка») задаётся атрибутом `style="--i:N"` у `<span class="l">`,
     где `N` — число символов отступа.
2. **Добавить стих** — проще всего добавить страницу в кэш-источник и перегенерировать
   (см. ниже), либо скопировать существующий `index.html` и поправить.

## Перегенерация из исходника (опционально)
В папке `tools/`:
- `crawl.py` — скачивает страницы Google Sites в `tools/cache/` (одноразово).
- `build.py` — собирает весь сайт из кэша.
- `struct.json` — порядок стихов в сборниках (для листания prev/next и оглавлений).

```bash
python3 tools/build.py
```

## Публикация
Любой коммит в ветку `main` автоматически пересобирает GitHub Pages.

```bash
git add -A && git commit -m "…" && git push
```

## Подключение домена shelamov.ru
См. инструкцию, выданную при настройке (DNS-записи A/AAAA для апекса и CNAME для www,
затем привязка домена в Settings → Pages и включение HTTPS).
